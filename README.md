# dotfiles

my dot files with [chezmoi](https://www.chezmoi.io)

## Usage

```bash
# install
curl -fsLS bit.ly/dot-ini | bash
# update completion
hash -r
# dry-run
chezmoi apply -vn
# apply
chezmoi apply -v
```
